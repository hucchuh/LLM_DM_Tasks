# V4 Clean WTP Trust Game

V4 is a cleaner follow-up to V3. It removes the most obvious demand cues and turns the main dependent variable into a costly behavioral choice.

Core changes:

- Yoked histories: the same numeric history is copied across language frames; only the partner message changes.
- Counterbalanced stake order: low, medium, and high stakes appear in different temporal positions.
- Exogenous language: messages do not depend on the previous return.
- One next stake per trial: the model is not asked to estimate low/medium/high return means.
- Behavioral dependent variables: continue choice, willingness to pay, next investment, and trust rating.

Main question:

> When behavior evidence is held constant, does cheap talk change costly cooperation with the partner?

Repeated Trust Game: LLM 如何更新对 partner 的信任？

两轮 MiniMax-M2.7 pilot。V1 先看自然差异条件下模型是否跟随实际返还；V2 把所有 partner type 的平均返还率控制为 0.40，再看模型是否还能区分行为结构。

核心结论：V1 中 trust_rating 几乎随 observed return 线性变化；V2 中平均返还率被完全控制后，模型仍能用 predicted return 读出 stake-specific pattern。也就是说，模型主要看行为，但不只是看总体平均收益。

Protocol

Game

模型与同一个 partner 连续互动。每轮模型投资 tokens，投资额会 tripled 后交给 partner，partner 决定返还多少。模型目标是最大化自身收益。

Readouts

trust_rating 测总体信任；predicted return 测模型预期 partner 会返还 tripled investment 的比例；investment 测最终行动。

V1: 自然差异条件

不同 partner type 同时改变语言线索和实际返还行为。问题是：模型信任评分更像是在听语言，还是在追踪实际返还？

V1 结论：模型不是主要被 friendly language 或 apology 牵引；trust_rating 很大程度跟随实际返还。但 V1 不能排除一个简单解释：模型可能只是追踪 average return。

V3: Noisy Behavior × Language Frame

V3 在 V2 的基础上加入噪声，并把 behavior pattern 与 language frame 正交化。所有 observed return 仍然等于 0.40，但同一个 behavior pattern 会分别配 neutral、warm promise、apology/excuse 三种语言。

Interpretation

Behavior-based
trust follows revealed returns more than cheap talk.
What V2 Adds
Not just mean
with average return controlled, structure still matters.
What V3 Adds
Language is small
warm/apology shifts trust modestly, not predicted return.

Next Step

1. Held-out prediction只给前几轮，让模型预测未展示的后续 trial。
2. Baselines比较 overall mean、stake-specific mean、recency-weighted mean。
3. Multi-model扩展到 GPT、Claude、Gemini、Qwen/Llama。
4. Human comparison比较人类和 LLM 是否同样主要依赖 revealed behavior。

V9 trial-level model

用现有 sequential 数据检验：模型投资主要由 payoff、return history、truth history，还是 instruction 驱动。

结论先行

  • previous payoff 是最稳的 trial-level predictor。 在 main partner 条件中，标准化系数 2.0982，cluster SE 0.3992。
  • cumulative return 也有正向作用。 标准化系数 0.6224，cluster SE 1.309。
  • cumulative truth 本身不稳定。 标准化系数 0.3748，接近 0。这和总相关 r=-0.031 一致。
  • instruction 会重写 honesty effect。 categorical model 里，low_honesty 系数 -1.9608；low_honesty × explicit_orthogonality 系数 7.3431。这说明 explicit orthogonality 不是中性说明，更像一种 causal-rule cue。

p 值为正态近似，标准误按 run_id 聚类。这个报告用于快速机制判断，不替代之后更正式的混合效应模型。

如何理解 explicit 反转

这里的关键不是“低诚实更可信”，而是 explicit orthogonality instruction 改变了模型的任务表征。原始提示词中的关键句是：

Important rule: the truth or falsehood of the pre-decision statement has no causal relation to the partner's return policy. Return behavior is generated independently of statement truth.

这句话可能起到类似 causal / de-biasing cue 的作用：它让模型停止把陈述真假当作合作性的社会证据，转而把任务看成一个需要通过投资来学习的收益过程。

需要特别区分两个概念：V9 控制的是 return_rate_t 或 return policy，而不是 realized payoff。真正拿到的 payoff_t = 10 - investment_t + returned_tokens_t 会被模型自己的投资额影响。因此早期投资差异会产生反馈回路：早投一点并赚钱，就更容易继续投；一直投 0，就永远看不到正收益证据。

在 fair/high + explicit 里，low-honesty 相比 high-honesty 的平均投资差为 5.222；6 个 paired seeds 全部同方向，简单 sign test 双侧 p = 0.03125。所以它不像单个 outlier，但仍可能是 early exploration / no-learning trap 被放大的路径效应。

V9b instruction-control

为了检验 explicit 反转是不是“多了一段正式说明”造成的，我们补了 attention-control：它同样增加一段正式说明，但不提 truth 与 return 的因果无关。结果显示 natural 和 attention-control 都是正向 high-low gap，而 explicit orthogonality 是强反向 gap。

讨论沉淀与 next step

Forced exploration 不是直接证明 risk-seeking 的工具。它更像一个必要的清洁步骤：先排除“早期不投资导致没有收益证据，从而形成 no-learning trap”这个朴素解释。

更干净的下一版可以让前 3 轮固定投资 5，或者给两组同样的外生 payoff observation，然后只分析第 4 轮之后的自由投资。如果 forced phase 后 explicit 低诚实优势消失，说明 V9 的反转主要是早期探索差异被 payoff feedback 放大；如果优势仍然存在，才更支持“low honesty + explicit orthogonality 改变了模型的 exploration / risk attitude”。

更严格的参数模型可以写成 causal-gated reinforcement learning：投资由预期返还 μ_t、诚实信念 h_t、honesty 的因果诊断权重 d_t、探索 bonus 和风险厌恶共同决定。最关键的待检验参数是 β_X：在 explicit 条件下，低诚实是否额外提高探索倾向。
